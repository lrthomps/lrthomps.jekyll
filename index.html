Random places to start

The papers suggested and chosen in the Vancouver DS reading group that takes place downtown on alternate Weds.

Common Datasets

Image sets (labelled):

  • MNIST: the hello world of machine learning; 25x25, centered, B&W handwritten digits. A good place to get started with easy wins to be made. Papers still publish new algorithms "benchmarked" with MNIST: think of it more as a proof of concept, not proof of better models. Papers should still include more complex datasets to make meaningful statements about any "advances" their work introduces.
  • CIFAR10/100: 32x32 colour images in 10/100 classes, with 6000/600 images per class. Stop here after MNIST. Download either from the source as pickle files or use tensorflow.keras.datasets.
  • ImageNet: with ~500 images per noun category of WordNet, this image set is close to or already overfit by researchers always benchmarking to it
